Premiers pas

(! Pour agrandir le texte, cliquez dessus)


L'interraction avec le programme est simple mais ne marche que sur Windows. (10 par exemple ).

Pour commencer, double-cliquez sur l'icône qui se trouve dans le premier dossier (juste avant le dossier nomé programme

Une fois cela fait cette fenêtre (ou une similliare en fonction du Windows ) s'affichera.

Appuyez une fois sur enter puis attendez qu'une fenêtre s'ouvre , vous ne pourrez rien faire pendant ce lapsus de temps là.

Le Menu principal

⇛ Aide ⇒ permet de connaitre les comandes, possibilités du programme

⇛ Recommencer ⇒ permet de néttoyer l'invite des comandes et de recommencer à 0, (cela revient à arrêter le programme t le rellancer, sans avoir à retrouver le fichier de démarage).

⇛ Procédure ⇒ vous indique la procédure à adopter pour pouvoir installer une application. Elle donne aussi des solutions si la méthode habituelle ne fonctionne pas.

⇛ Installer Rastop et ses composants ⇒ script qui vous permet d'installer facilement rastop eet une certaine quantité de molécules dans votre ordinateur pour qu'après vous puissiez les visualiées.

⇛ Installation intéractive ⇒ Ce boutton vous redirige ver une autre partie du programme qui vous permettra de facilement installer votre programme sans problème.

⇛ Quitter Ce boutton vous permet d'arrêter correctement le programme.

Après avoir cliqué sur le boutton Installer rastop et ses composants voici le menu qui ous sera rpoposé:

⇛ rastop ⇒ Ce bouton vous permettra d'installer rastop, raswin ou rasmol en fonction de votre choix sur le menu suivant.

⇛ ses composants ⇛ Ce boutton vous proposera un large choix de molécules que vous pourrez facilement installer.

⇛ sortir ⇒ Ce boutton vous permet de sortir de ce menuet de retourener au menu principal.

Après avoir cliqué sur le boutton rastop vous aurez un dernier menu, celui-ci vous proposera trois options:

⇛ rastop ⇒ Ce Boutton vous permettra d'installer rastop.

⇛ rasmol ⇒ Ce boutton vous permettra d'installer rasmol.

⇛ sortir ⇒ Ce boutton vous permet de sortir de ce menu pour retourner au choix entre l'installation des applications ou des molécules.
